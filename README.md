# Проект Магазина Одежды

Этот проект представляет собой веб-сайт онлайн-магазина одежды. Сайт адаптивен и отлично просматривается на устройствах с разными разрешениями экрана.

## Стек

- HTML
- CSS / SCSS
- JavaScript
- Bootstrap
